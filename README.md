# Getting Started

Vaadin needs NodeJS 10.x or later to generate the front-end resource bundle. You can install NodeJS locally to your current project by using the following Maven command:
mvn com.github.eirslett:frontend-maven-plugin:1.7.6:install-node-and-npm -DnodeVersion="v10.16.3"
